# Venera

## 安装

### 从 Release 下载

从 [GitHub Releases](https://github.com/haukuen/venera/releases) 下载最新版本。

### 包管理器下载

**Windows (Scoop)**

```bash
scoop bucket add endless https://github.com/haukuen/endless
scoop install venera
```

**macOS (Homebrew)**

```bash
brew tap haukuen/tap
brew install --cask venera
```
